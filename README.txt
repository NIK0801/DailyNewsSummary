config.py - содержит словарь из ссылок на ленты RSS новостей и 2 пересекающихся вектора для выборки; путь к файлам allnews.csv,
где содержатся все новости и certainnews.csv - отобранные по векторам

parseEconomNews.py 

	check_url() - парсим rss
	take_data() - достаем нужные данные и составляем словарь, записывая только новости за последние сутки, возвращаем словарь
	write_al_news() - сохраняем в csv файл
	looking_for_certain_news() - фильтруем по векторам при помощи лямбда выражений, сохраняем в формате .html в папку db

sendToEmail.py
	
	отправка файла .xlsx на почту, более подробное описание функций в самом файле

removeOldFiles.py

	удаляет файлы .xlsx старше 2х недель из папки db



Скрипт  sendToEmail.py добавлен в Cron с расписанием 13:59 каждого дня.
Если добавляем скрипт в локальный Cron, то пользователь должен иметь все права доступа к вышеперечисленным файлам и быть их владельцем

chmod +x file.py - даем права на выполнение
chown -R user:user file.py - задаем владельца
