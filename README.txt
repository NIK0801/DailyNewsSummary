DailyNewsSummary - средство сбора экономических новостей за сутки(можно установить любой другой промежуток времени), парсит RSS-ленты сайтов, потом прогоняет эти новости по 2м векторам из слов посредством лямбда-функций, сохраняет отобранные новости в файл .html формата и отправляет по указанной почте.
Почта, с которой идет отправка, заведена на рабочем телефоне, можно использовать Mail или Gmail, в скрипте указать этот адрес, вместо пароля 

config.py - содержит словарь из ссылок на ленты RSS новостей и 2 пересекающихся вектора для выборки; путь к файлам allnews.csv,
где содержатся все новости и certainnews.csv - отобранные по векторам
ссылки на ленты rss это не просто url сайта, чтобы найти эту ссылку заходим на интересующий нас новостной сайт, переходим в раздел "новости" и переходим в режим просмотра кода страницы комбинацией клавиш ctrl+U, затем нажимаем F3 чтобы открылась строка поиска по странице и пишем в этой строке "rss" находим соответствующую ссылку, где фигурирует эта rss, жмем на нее - должна открыться новостная лента rss и уже URL этой страницы копируем и вставляем в config.py

parseEconomNews.py 

	check_url() - парсим rss
	take_data() - достаем нужные данные и составляем словарь, записывая только новости за последние сутки, возвращаем словарь
	write_al_news() - сохраняем в csv файл
	looking_for_certain_news() - фильтруем по векторам при помощи лямбда выражений, сохраняем в формате .html в папку db

sendToEmail.py
	
	отправка файла .html на почту, более подробное описание функций в самом файле

removeOldFiles.py

	удаляет файлы .html старше 2х недель из папки db



Скрипт  sendToEmail.py добавлен в Cron с расписанием 13:59 каждого дня.
Если добавляем скрипт в локальный Cron, то пользователь должен иметь все права доступа к вышеперечисленным файлам и быть их владельцем

chmod +x file.py - даем права на выполнение
chown -R user:user file.py - задаем владельца
